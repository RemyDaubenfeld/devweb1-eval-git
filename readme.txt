Evaluation GitHub - DevWeb1 - 10/01/2024

1 - Créer un dépôt git local "devweb1-eval-git"
2 - Ajouter un fichier "readme.txt" dans lequel vous repertorirez TOUTES les commandes git saisies
3 - Faites un commit à chaque commande ajouté
4 - Déployez votre projet sur GitHub
5 - Créer une branche dev avec un fichier "index.html" dans lequel vous ferez figurer la liste  de toutes le communes de France que vous déploierez  sur GitHub.

Liste des commandes:

git init
git add .

création du dépôt distant sur GitHub

git remote add origin https://github.com/RemyDaubenfeld/devweb1-eval-git.git
git branch -M master
git push -u origin master

git commit -a -m "First commit"

git push -u origin master

Modification du document readme.txt puis "ctrl + S"

git commit -a -m "Second commit"
